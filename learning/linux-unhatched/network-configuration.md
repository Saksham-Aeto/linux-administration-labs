# Commands Learned

## `ifconfig`

Stands for Interface Configuration.

Used to display network configuration information.

**Syntax:**

```bash
ifconfig [OPTIONS]
```

---

# Notes

- Can temporarily modify network settings.
- Mostly used to display network information.
- Permanent changes are usually done using configuration files.

---

## `iwconfig`

Similar to ifconfig but dedicated to wireless network interfaces.

---

# Key Takeaways

- ifconfig shows network interface information.
- iwconfig is used for wireless interfaces.
- Network settings changed by ifconfig are usually temporary.
