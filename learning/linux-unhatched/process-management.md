## Objective

Learn how Linux processes work and how to view and manage running processes.

---

## Important Concepts

- A running command becomes a **process**.
- Processes run with the privileges of the user that started them.
- Normal users cannot interfere with other users' processes.
- Root can control all processes on the system.
